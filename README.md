In this assignment, you are required to design and develop a Company Registration / Service Request Form using pure HTML only.
The purpose of this task is to assess your understanding of:
HTML form structure
-Input types
-Form validation
-Proper use of labels
-Grouping elements using fieldset and legend
-Clean and readable code formatting

Important: No CSS and No JavaScript are allowed.
Learning Objectives
By completing this assignment, students should be able to:
Create a well-structured HTML form
Use appropriate input types such as email, tel, url, and date
Apply HTML validation attributes correctly
Associate <label> elements properly with form inputs
Organize forms into logical sections
Assignment Requirements
Your form must include the following sections:

1. Personal Information
Full Name
Email Address
Phone Number

2. Company Information
Company Name
Company Website
Company Size (Select dropdown)
Industry

3. Service Request
Service Type (Radio buttons)
Preferred Contact Method (Checkboxes)
Appointment Date
Appointment Time
Budget Range

4. Message and Agreement
Textarea for additional information
Required Terms and Conditions checkbox
Technical Requirements
File name must be: index.html
Use <form method="post" action="#">
Use at least two <fieldset> elements
Every input must have a corresponding <label>
Use proper validation attributes including:
required
min, max
minlength, maxlength
pattern
Include both:

Submit button

Reset button
